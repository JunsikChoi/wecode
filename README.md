# WECODE GIT PRACTICE

## GIT COMMANDS

### git remote <remote-name> <remote-url>
git 저장소의 원격 저장소 위치를 remote-name에 할당한다.

### git add <file-to-add>
변경 사항을 커밋하기 전에 추가한다.

### git commit -m <commit-msg>
변경 사항을 커밋한다. 커밋 메세지 추가 가능

### git branch
현재 있는 깃 브랜치를 조회한다.

### git checkout <branch-name>
<branch-name>으로 브랜치를 변경한다. 변경하기 전 변경 내용은 커밋 후 옮기는게 좋다.
